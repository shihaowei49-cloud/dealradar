## 四、自动更新流程（搭建后）

```
每天 08:00 (北京时间)
    ↓
GitHub Actions 触发 fetch_deals.py
    ↓
v2ex API + HN API 更新数据
    ↓
commit deals.json 到 GitHub
    ↓
Cloudflare Pages 自动重新部署
    ↓
用户刷新页面即可看到最新数据
```

也可在 GitHub → Actions → 每日数据更新 → **Run workflow** 手动触发。

## 五、关键技术备注

- deals.json fetch 请求自带时间戳参数防缓存：`./data/structured/deals.json?<timestamp>`
- 前端核心函数：`load()`（数据加载）、`enrichDeal()`（评分计算）、`renderAll()`（渲染）
- 评分权重涉及：来源权重、分类权重、社交热度（views/likes/replies）、帖子新鲜度（ageDays）
- localStorage 键名：`isWhitelist`、`__not_first_visit__`、`loglevel`
