```

├── fetch_deals.py                    ← 每日数据更新爬虫脚本

```

---

## 第一步：在 GitHub 新建仓库

1. 打开 https://github.com/new
2. 仓库名填 `dealradar`（或任意名称）
3. 其余保持默认，**不要勾选 Initialize README**
4. 点击 **Create repository**
5. 复制页面上显示的仓库地址，格式如：
   ```
   https://github.com/你的用户名/dealradar.git
   ```

---

## 第二步：推送代码

双击运行 `setup_git.bat`，脚本会自动：
- 清理旧 `.git` 残留
- 初始化仓库并提交所有文件
- 提示你输入仓库地址后推送到 GitHub

> **注意**：首次推送 GitHub 会弹出登录验证，用浏览器授权或输入 Personal Access Token 即可。

---

## 第三步：连接 Cloudflare Pages

1. 打开 https://dash.cloudflare.com → Pages → Create a project
2. 选择 **Connect to Git** → 授权 GitHub → 选择 `dealradar` 仓库
3. 构建设置：
   - Framework preset: `None`
   - Build command: 留空
   - Output directory: `/`（根目录）
4. 点击 **Save and Deploy**

之后每次推送到 GitHub，Cloudflare Pages 会自动重新部署。

---

## 第四步：GitHub Actions 每日自动更新

`.github/workflows/daily-update.yml` 已配置好，每天北京时间 **08:00** 自动运行：

```
爬虫脚本采集数据 → 更新 deals.json → 自动 commit → Cloudflare Pages 重新部署
```

**支持的数据源：**
| 来源 | 方式 | 状态 |
|------|------|------|
| v2ex | 官方 API | ✅ 全自动 |
| Hacker News | Firebase API | ✅ 全自动 |
| linux.do | Discourse API（需登录） | ⚠️ 未登录时可能受限 |
| nodeseek | 需浏览器 | ⚠️ Actions 中跳过 |
| 52pojie | 访问被拦截 | ❌ 暂不支持 |

---

## 手动触发更新

在 GitHub 仓库页面 → **Actions** → **每日数据更新** → **Run workflow**，可随时手动跑一次。

---

## 网站前端新增功能

在 `index.html` 右上角已注入代码逻辑，可添加 **↻ 获取最新数据** 按钮。

将 `index.html` 中 topbar 部分改为：

```html
<!-- 找到这行 -->
<div class="updated" id="updated">Loading</div>

<!-- 替换为 -->
<div style="display:flex;align-items:center;gap:10px">
  <div class="updated" id="updated">Loading</div>
  <button class="btn" id="refresh-btn" onclick="refreshData()" style="height:30px;font-size:12px">↻ 获取最新数据</button>
</div>
```

并在 JS 的 `load()` 函数旁边添加：

```js
async function refreshData(){
  const btn = $('#refresh-btn');
  btn.disabled = true;
  btn.textContent = '获取中…';
  await load();
  btn.disabled = false;
  btn.textContent = '↻ 获取最新数据';
}
```
